> Leading supplier of professional scientific instruments, laboratory equipment, calibration tools, and maintenance services in Agadir, Morocco.

Univers Instrument Service provides high-quality equipment for research, industry, and education. This modern web application serves as a comprehensive catalog and information portal for clients across Morocco.

## ✨ Features

- **🌍 Multi-language Support**: Fully localized in English, French, Arabic, and Spanish.
- **📦 Comprehensive Catalog**: Browse products across various categories:
  - Laboratory & Scientific Instruments
  - Medical Equipment
  - Agricultural & Water Analysis Tools
  - Industrial Chemicals & Consumables
  - Laboratory Furniture
- **📱 Premium Responsive Design**: Optimized for all devices with a focus on ease of use.
- **⚡ Performance First**: Built with Next.js 16 for lightning-fast page loads and SEO.
- **🔍 Advanced SEO**: Integrated JSON-LD structured data and optimized meta tags for search engine visibility.
- **🌑 Light/Dark Mode**: Adaptive interface for comfortable viewing in any environment.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) with [Zod](https://zod.dev/) validation
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or pnpm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/univers-instrument.git
   cd univers-instrument
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
